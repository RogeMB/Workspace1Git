OVERVIEW EJERCICIO 04

Este paquete contiene un programa sencillo para generar un ticket de productos de un supermercado, con las siguientes clases:

  • Alimentación
  • Ropa
  • Electrónica
  • Producto
  • Venta
  • Línea de venta

Además una clase principal para comprobarlo todo.

En cuánto a métodos, se podrá calcular el precio unitario de un producto además del subtotal. Se podrá aplicar un descuento si el producto de alimentación está proximo a caducar y un extra en el precio si es de electrónica. Además de métodos para imprimirlo todo.

Todo ha sido generado desde JDK 11. Cualquier versión anterior puede generar erróres.
